# Readme

## Install

```
$ git clone git@github.com:nickahoy/realitycheck.git
$ virtualenv venv
$ . venv/bin/activate
$ pip install --editable .
```

## Run

```
$ reality_check
```

## Caveats

**Local Property Tax** 
- Local Adjustment Factor not taken into account
- Assumes you're paying with a mortgage
